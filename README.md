# 本地照片对比工具

这是一款专注隐私的本地图片对比应用，适合对比修图前后、版本差异或现场拍摄的多张方案。所有操作都在浏览器内完成，不依赖云端上传。

## 你可以用它来

- 快速创建多个分组，按项目或场景整理待对比的图片
- 将图片拖拽或选择导入，随时设为 A/B 进行分屏滑杆对比
- 在图片库里切换、删除或重新指派 A/B，右侧视图即时更新
- 调整侧栏与缩略图区的宽度或折叠面板，腾出更多对比空间

## 如何开始

1. 浏览器打开项目后，点击左侧「添加图片」或直接拖拽文件到图片列表。
2. 为图片选择「设为 A」和「设为 B」，右侧会显示可拖动滑杆的对比视图。
3. 通过左侧分组切换不同项目；需要时可重命名或删除分组（会同时清理其中的图片与对比状态）。
4. 缩略图读取失败会提示「文件不可访问」，可重新选择或移除该图片。

## 隐私与存储

- 数据仅存储在本地：状态保存在浏览器的 localStorage，文件句柄或文件 Blob 写入 IndexedDB，方便下次直接读取。
- 支持 File System Access API；若浏览器不支持，会自动回退到隐藏的文件选择器，同样保证可以导入和持久保存。
- 读取文件前会按需申请权限；拒绝后可以随时重新授权或删除记录。

## 运行与兼容性

- 推荐使用支持 File System Access API 的现代 Chromium 浏览器，能获得最佳的文件读取与拖拽体验。
- 若需本地预览，可执行 `pnpm install && pnpm run dev` 启动开发服务器后在浏览器访问显示的地址。
- 构建、测试与代码检查命令：`pnpm run build`、`pnpm test`、`pnpm run lint`。
